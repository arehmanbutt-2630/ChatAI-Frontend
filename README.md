# aichat-frontend

## Project Setup

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd aichat-frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open the app:**

   Visit [http://localhost:5173](http://localhost:5173) in your browser.
